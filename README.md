📚 Online Bookstore

A modern Android application built with Kotlin that allows users to browse books, search for titles, manage their cart, and enjoy a smooth online bookstore experience.

✨ Features

- 📖 Browse books
- 🔍 Search books
- 👤 User authentication
- 🛒 Shopping cart
- ❤️ Wishlist
- 📱 Clean and user-friendly interface

🛠️ Tech Stack

- Language: Kotlin
- IDE: Android Studio
- Build Tool: Gradle
- Version Control: Git & GitHub

🚀 Getting Started

1. Clone this repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Run the app on an Android emulator or physical device.
